# Mqrb-cli

CLI of Monocoque-ruby

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'mqrb'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install mqrb

## Usage

```bash
mqrb create-app myapp --runtime=mqrb-with-compiler
```
